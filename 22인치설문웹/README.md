# 22인치 터치 설문 키오스크 데모 (NEXA One)

오프라인 행사장용 **설문 키오스크 웹 2종 + 실시간 통계 대시보드 + 관리자(엑셀 다운로드)** 데모.
요구사항 원문: [22인치 터치 모니터 맞춤 2종 설문 웹 프로그램 구축.txt](22인치%20터치%20모니터%20맞춤%202종%20설문%20웹%20프로그램%20구축.txt) · 레퍼런스: [레퍼런스.md](레퍼런스.md) · 요구사항 대조: [요구사항체크리스트.md](요구사항체크리스트.md)

가상의 제품 「NEXA One 스마트 공기청정기」 전시 부스를 가정했다.

## 실행

빌드/서버 불필요. `index.html`을 브라우저로 열면 된다.

| 화면 | 파일 | 요구사항 대응 |
|---|---|---|
| 런처 | `index.html` | (데모 안내용) |
| 키오스크 1 — 일반 설문형 | `kiosk1.html` | 대기 화면 → **설문 동의 모달** → 객관식 4문항 순차 노출 → 감사 화면 → 7초 뒤 자동 복귀, 무응답 60초 시 초기화 |
| 키오스크 2 — 정보 제공+설문형 | `kiosk2.html` | 특장점 6종 탭/스와이프 UI(8초 자동 순환, 특장점별 전용 이미지), 단일 질문 투표, 응답 직후 실시간 결과(비율·투표수) 노출 |
| 실시간 대시보드 | `dashboard.html` | 키오스크 1·2 데이터 실시간 수합, KPI + 문항별 막대 차트 |
| 관리자 | `admin.html` | **로그인 게이트(데모: admin / nexa2026)**, 전체 응답 테이블, 키오스크별 필터, 엑셀(CSV, UTF-8 BOM) 다운로드, 초기화 |

## 구현 특징 (요구사항 매핑)

- **1920×1080 해상도 고정형**: `.stage` 고정 캔버스를 창 크기에 맞춰 자동 스케일 — 22인치 FHD 모니터에서 1:1 표시
- **화면 제어**: 우클릭/드래그/더블클릭/핀치줌/스크롤/F5·Ctrl+R 차단 (`NX.lockKiosk`), 전체화면 버튼 제공
- **데이터 관리**: 데모는 `localStorage` 저장. 탭 간 `storage` 이벤트 + 2초 폴링으로 실시간 동기화 → 키오스크와 대시보드를 별도 창으로 열고 시연
- **시연 데이터**: 첫 접속 시 응답 109건 자동 생성, 대시보드/관리자에서 "응답 5건 시뮬레이션" 버튼으로 실시간 유입 연출
- **비주얼 에셋**: 제품·특장점 이미지 7종은 힉스필드(Higgsfield) AI로 생성 (`assets/img/`)
