# FORME — AI 핏 커머스 풀사이트 프로토타입

정적 HTML/CSS/JavaScript로 만든 전면 실구동형 패션 커머스 프로토타입입니다. 상품 탐색, 검색, 필터, 찜, AI 핏 추천, 장바구니, 데모 결제, 주문 완료, 주문 내역, 핏 프로필을 하나의 상태 모델로 연결합니다.

## 실행

```bash
npm install
npm run serve
```

브라우저에서 `http://127.0.0.1:4173`을 엽니다. UI 프리미티브는 `/showcase.html`에서 확인할 수 있습니다.

## 검증 흐름

1. 상품 상세에서 `내 사이즈 찾기` 선택
2. 키, 몸무게, 선호 핏 입력
3. 추천 결과에서 `M 사이즈 적용하기` 선택
4. 상품 옵션이 M으로 바뀌고 구매 버튼이 활성화되는지 확인

자동 검증은 `npm test`로 실행합니다. 상품 이미지·모션을 Higgsfield로 제작할 때는 `HIGGSFIELD_ASSET_BRIEF.md`의 파일명, 가상 모델 정책, 일관성 규칙을 사용합니다.

## 전체 구매 여정

홈·Shop 상품 탐색 → 검색·카테고리 필터 → 상품 상세 → 핏 추천 → 추천 사이즈 적용 → 장바구니 → 배송·데모 결제 → 주문 완료 → 마이페이지 주문·핏 프로필 확인.

상품·장바구니·회원·주문·핏 프로필은 브라우저 저장소에 유지됩니다. 외부 결제, 서버 인증, 택배 및 실제 재고 연동은 프로토타입 상태로 시뮬레이션합니다.

상품 이미지 6종은 인증된 Higgsfield MCP의 `recraft_v4_1` 모델로 생성했으며 인물을 사용하지 않았습니다. 향후 인물 이미지가 필요한 경우 가상 성인 모델 정책을 따릅니다.
