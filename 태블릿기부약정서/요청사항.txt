업무 내용
[프로젝트 개요]
- 대학교 캠퍼스 발전기금 전자 기부약정서 PWA 개발

[프로젝트 배경 및 목표]
- 현재 종이 기부약정서를 작성하고 보관하는 방식을 태블릿 기반으로 전환하여 모금 상담 시 업무 효율을 높이고자 합니다. 복잡한 기부관리시스템이나 결제 연동 없이, 종이 약정서를 디지털화하여 태블릿에서 직접 정보를 입력받고 서명을 거쳐 PDF 파일로 자동 생성 및 저장하는 매우 단순하고 직관적인 전자문서 시스템 구축이 목표입니다.

[과업 범위]
1. 수행 범위
- 상세 기획: 입력 항목 매칭, PDF 출력 포맷 매핑
- UI/UX 디자인
- 프런트엔드/Client 개발
- 백엔드 개발 (파일 저장 및 관리자 페이지 연동용)

2. 상세 기능 요구 사항
2-1. 약정서 입력 폼
- 기부자 정보, 연락처, 주소, 구분, 기부금액, 기부용도, 납부방법 데이터를 화면에서 입력
- 기존 사용 중인 종이 기부약정서와 최대한 동일한 형태의 직관적인 UI 구성
- 태블릿 기기 환경에 맞춘 화면 렌더링 제공

2-2. 전자 서명
- 손가락, 스타일러스 펜을 인식하는 서명 입력 캔버스 영역 제공
- 작성 중인 서명 초기화, 다시쓰기 기능 제공
- 서명 완료된 데이터를 투명 배경의 이미지로 변환하여 유지

2-3. 통합 및 PDF 생성
- 약정서 폼에 입력된 텍스트 데이터, 변환된 서명 이미지를 수신하여 하나의 문서로 통합
- 통합된 문서를 원본 HWP 약정서와 동일한 레이아웃의 PDF 파일로 자동 생성
- 생성된 파일명을 20260629_홍길동_10000000.pdf 와 같은 지정 규격으로 자동 변경

[기술/제조 스택]
- 제안사에서 비용을 최소화하고 가장 빠르게 구축할 수 있는 PWA 프레임워크 및 오픈소스 PDF 생성 라이브러리 자유 제안

[지원 자격 및 우대 사항]
1. 지원 자격
- 웹 브라우저 환경에서 전자서명 및 PDF 변환 기능 구현 경험
2. 우대 사항
- 복잡한 솔루션 도입 없이 제안 예산 내에서 가볍고 빠르게 목적을 달성할 수 있는 파트너

[산출물]
- 소스코드 원본
- 디자인 및 기획 파일
- 관리자 페이지 접속 정보
- 사용자 매뉴얼
- 설치 및 운영 가이드

[계약 관련 특이 사항]
- 오직 태블릿에서 양식을 채우고 서명한 뒤 PDF로 저장하여 관리자가 다운로드할 수 있게 만드는 핵심 기능에만 집중합니다. 협의된 예산에 맞추어 가장 경제적이고 효율적인 기술 구현 방안을 제안해 주시기 바랍니다.
