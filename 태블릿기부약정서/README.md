# 한울대학교 발전기금 전자 기부약정서 (PWA 데모)

종이 기부약정서를 **태블릿 전자문서**로 전환하는 경량 PWA 데모입니다.
모금 상담 현장에서 정보를 입력받고 → 손글씨 서명을 받고 → 원본과 동일한 레이아웃의 **PDF를 자동 생성·저장**하고 → 관리자가 다운로드하는, 요청사항의 핵심 기능에만 집중했습니다.

## 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 데모 소개 랜딩 (이용 흐름·기능·기술 구성) |
| `app.html` | **전자 약정서 작성** — 4단계 폼 · 전자서명 · PDF 생성 (핵심) |
| `admin.html` | **관리자 페이지** — 접수 목록·검색·통계·PDF 미리보기/다운로드·CSV |
| `manifest.json` · `sw.js` | PWA 매니페스트 · 서비스워커(오프라인 캐싱) |
| `icons/` | 앱 아이콘 (192/512) |

## 요청사항 대응

- **2-1 약정서 입력 폼** — 기부자 정보·연락처·주소·구분·기부금액·용도·납부방법을 종이 양식과 동일한 항목/순서로 입력. 한글 금액 자동 변환, 금액 프리셋, 태블릿 터치 최적화, 필수값 검증.
- **2-2 전자 서명** — 손가락·스타일러스 인식 캔버스(Pointer/Touch 이벤트). 되돌리기·다시쓰기. 여백 트리밍 후 **투명 배경 PNG**로 변환·유지.
- **2-3 통합 및 PDF 생성** — 입력 텍스트 + 서명 이미지를 원본 A4 레이아웃 템플릿에 통합해 PDF 생성. 파일명 `YYYYMMDD_이름_금액.pdf` (예: `20260629_홍길동_10000000.pdf`) 자동 규격화.
- **백엔드/관리자 연동** — 데모는 브라우저 저장소로 백엔드를 시뮬레이션. 관리자 페이지에서 목록·PDF 다운로드. 운영 시 Supabase Storage 등으로 손쉽게 확장.

## 기술 스택 (비용 최소·빠른 구축)

- **PWA (Vanilla JS)** — 프레임워크 빌드 없이 정적 호스팅만으로 배포
- **jsPDF** — 브라우저에서 PDF 즉시 생성 (오픈소스)
- **html2canvas** — HTML 약정서 템플릿을 이미지로 래스터화 → 한글 폰트 임베딩 이슈 없이 원본 레이아웃 재현
- **Canvas 서명 / Service Worker** — 전자서명·오프라인 캐싱

## 실행

정적 파일이라 서버만 있으면 됩니다.

```bash
cd 태블릿기부약정서
python -m http.server 8080
# http://localhost:8080/  (또는 /app.html 바로)
```

- 관리자 데모 계정: **admin / 1234**
- 관리자 화면 상단 **“데모 데이터 채우기”**로 샘플 목록 확인 가능
- 데이터는 브라우저 `localStorage`에만 저장됩니다 (데모용).

## 운영 전환 시 (참고)

1. 제출 시 PDF 데이터 URI를 **Supabase Storage**(또는 S3)에 업로드
2. 약정 메타데이터(이름·금액·용도·납부·서명URL)를 DB 테이블에 저장
3. 관리자 페이지는 목록 API를 조회하고 서명된 다운로드 URL 제공
4. 로그인은 실제 인증(관리자 계정)으로 교체
