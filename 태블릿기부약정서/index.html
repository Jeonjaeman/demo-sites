HOW IT WORKS

약정부터 저장까지 4단계

상담 직원은 태블릿을 건네주고, 기부자는 화면 안내에 따라 작성만 하면 됩니다.

01

기부자 정보 입력

성명·연락처·주소·소속 등 종이 약정서와 동일한 항목을 태블릿에 최적화된 폼으로 입력합니다.

02

기부 내용 지정

기부금액(한글금액 자동 변환)·용도·납부방법을 선택합니다. 자주 쓰는 금액은 버튼 한 번으로.

03

동의 · 전자서명

개인정보 약관 동의 후, 손가락·스타일러스로 직접 서명. 서명은 투명 배경 이미지로 저장됩니다.

04

PDF 자동 생성

입력값과 서명을 원본 양식에 통합해 PDF로 생성하고 날짜_이름_금액.pdf로 자동 저장·전송합니다.

KEY FEATURES

요청하신 핵심 기능

“오직 태블릿에서 채우고 서명한 뒤 PDF로 저장하여 관리자가 다운로드” — 딱 그 목표에 맞췄습니다.

📝

종이 양식 그대로의 입력 폼

기존 종이 약정서와 동일한 항목·순서로 구성한 직관적 UI.

  • 단계별(스텝) 진행 · 필수값 검증
  • 한글 금액 자동 변환 · 금액 프리셋
  • 태블릿 터치 최적화 렌더링
✍️

전자 서명 캔버스

손가락과 스타일러스 펜을 모두 인식하는 서명 영역.

  • 되돌리기 · 다시쓰기 지원
  • 여백 자동 트리밍
  • 투명 배경 PNG로 변환·유지
📄

PDF 자동 생성 · 규격 저장

텍스트와 서명을 원본 레이아웃으로 통합해 PDF 출력.

  • A4 원본 약정서와 동일 레이아웃
  • 20260629_홍길동_10000000.pdf 규격 자동 파일명
  • 기기 저장 + 관리자 페이지 전송
🗂️

관리자 페이지

접수된 약정서를 목록으로 확인·관리.

  • 검색·필터 · 통계 대시보드
  • PDF 미리보기 · 다운로드
  • CSV 내보내기
📲

PWA · 오프라인 대응

홈 화면에 설치해 앱처럼 실행. 상담 부스 네트워크가 불안정해도 작성 가능.

  • 설치형 웹앱(매니페스트)
  • 서비스워커 캐싱
  • 별도 앱스토어 심사 불필요
🔒

개인정보 동의 · 예우

수집·이용 동의 절차를 약정 흐름에 내장.

  • 필수 약관 동의 게이트
  • 기부금영수증 발급 선택
  • 보관·파기 안내 문구

가장 가볍고 빠른 구현

제안 예산 안에서 가장 경제적인 오픈소스 스택으로 구성했습니다. 무거운 기부관리시스템·결제 연동 없이, 핵심 기능만 견고하게.

PWA (Vanilla JS) jsPDF html2canvas Canvas 서명 Service Worker Supabase(선택)
이 구성을 택한 이유
  • 프레임워크 빌드 없이 정적 호스팅만으로 배포 → 서버비용 최소
  • PDF를 브라우저에서 즉시 생성 → 서버 왕복·대기 없음
  • 파일 저장·관리는 Supabase Storage 등으로 손쉽게 확장
  • 본 데모는 브라우저 저장소로 백엔드를 시뮬레이션
