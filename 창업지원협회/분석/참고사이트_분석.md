# 참고 사이트 분석

> 창업지원협회 공식 홈페이지 데모 구축을 위한 벤치마킹 분석
> - **국민체육진흥공단(ksponco.or.kr)** → 공공기관 포털의 **디자인 · UI/UX · 인터랙션 · 스크롤 효과 · 롤링 배너** 레퍼런스 (집중 분석 대상)
> - **요청사항.txt(RFP)** → 콘텐츠 · 기능 요구사항 (공지/정보 게시판, 1:1 문의, 회원/마이페이지, 관리자 백오피스)

---

## 1. ksponco.or.kr (국민체육진흥공단) — 디자인/인터랙션 분석

### 1-1. 포지셔닝 / 톤앤매너
대국민 신뢰를 전달하는 **공공기관(공단) 포털**. "정보 전달의 신뢰성 + 깔끔한 시각 정돈"을 동시에 추구한다. 화이트 베이스 + 포인트 블루, 큼직한 영문 키워드(Creativity / Challenge / Passion / Future)와 한글 서브카피의 조합으로 **권위감과 모던함의 균형**을 잡는다.

### 1-2. 페이지 구조 (스크롤 순서)
1. **헤더 + GNB** — 상단 유틸(로그인/검색/사이트맵) + 가로형 메인 메뉴(정보공개·국민참여·소식·사업안내·경영공개·기관소개). 스크롤 시 헤더 축소/솔리드 전환.
2. **메인 비주얼(롤링 배너)** — 4개 패널이 자동 순환하는 **이미지 캐러셀**. 대형 배경 이미지 + 중앙 영문 키워드 + 한글 카피 + CTA 버튼. 좌/우 화살표, 하단 인디케이터(도트/넘버), 재생·정지 토글.
3. **퀵 메뉴(바로가기)** — 시설예약·오시는 길·사이버투어·고객제안·자주묻는질문 등 아이콘 숏컷 카드. 고정 사이드 퀵바도 운영.
4. **공지/소식 보드** — 공지사항·채용공고·보도자료 3종 게시판을 **탭/멀티칼럼**으로 노출. 보도자료는 썸네일 카드.
5. **사업 안내** — 주요 사업/서비스를 카드·배너로 안내.
6. **홍보 콘텐츠 / 배너 존** — 캠페인 배너, 관련 사이트 바로가기.
7. **푸터** — 로고 + 기관정보 + 멀티칼럼 사이트맵 + 관련사이트 드롭다운 + 웹 표준/인증 마크.

### 1-3. 컬러 / 타이포
- **컬러**: 클린 화이트 배경 + 다크 그레이 본문 + 포인트 블루 링크. 공공기관 특유의 절제된 채도.
- **타이포**: 산세리프 한글 본문 + 영문 키워드 대형 강조. 위계가 분명한 사이즈 스텝.
- **스타일**: 격식 있는 위계, 텍스트 기반 내비게이션, 접근성(웹 표준/인증) 강조.

### 1-4. 인터랙션 / 스크롤 효과 (데모에 구현할 목록)
- **롤링 배너 자동 순환** — N초 자동 전환, 인디케이터/화살표/재생·정지, 슬라이드 진입 시 카피 페이드·슬라이드업.
- **헤더 트랜스폼** — 최상단 투명 → 스크롤 시 솔리드/축소.
- **퀵 메뉴 호버 마이크로 인터랙션** — 아이콘 부양·컬러 전환.
- **섹션 진입 reveal** — fade-up / stagger.
- **숫자 카운트업** — 핵심 지표(지원 기업 수, 누적 멘토링 등).
- **공지 보드 탭 전환** — 공지/정보/보도 즉시 스위칭.
- **Back-to-top, 플로팅 상담** 버튼.

> 본 데모는 여기에 더해 **부드러운 스크롤(Lenis)** 과 **GSAP ScrollTrigger 기반 핀/스케일 브랜드 리빌·워터마크 패럴럭스**를 추가해, 정적 공공기관 톤에 시네마틱 모션을 결합한다(요청: 인터랙션·스크롤·애니메이션 집중).

## 2. 도메인 치환 — 국민체육진흥공단 → 창업지원협회

| 레퍼런스(공단) | 데모(창업지원협회) |
|----|----|
| 체육/스포츠 진흥 | **창업·스타트업 성장 지원** |
| 시설예약·오시는길 | **지원사업 신청·멘토링 예약·오시는 길** |
| 공지/채용/보도 | **공지사항 · 정보제공(창업뉴스) · 보도자료** |
| 블루 단색 포인트 | **블루(신뢰) + 에메랄드 그린(성장) 그라데이션** — 공공 신뢰감 + 스타트업 에너지 |
| 스포츠 비주얼 | **창업 허브/코워킹/IR 데모데이/글로벌 진출** (Higgsfield 생성 고품질 미디어) |

- **브랜드(가칭)**: 대한창업지원협회 / KOREA STARTUP SUPPORT ASSOCIATION — 영문 약칭 **KSSA**.
- **메인 키워드 슬라이드**: `CHALLENGE`(도전) · `GROWTH`(성장) · `CONNECT`(연결) · `GLOBAL`(세계로) — ksponco의 영문 키워드 패턴 차용.

| RFP 요구사항 | 데모 반영 페이지 | 비고 |
|----|----|----|
| 메인(롤링 배너, 퀵 메뉴, 최신 공지, 시각효과) | `index.html` | 롤링 배너 + 퀵메뉴 + 지원사업 + 통계 + 공지 프리뷰 |
| 협회 소개(인사말·연혁·오시는 길) | `about.html` | 인사말, 비전, 연혁 타임라인, 조직, 오시는 길(약식 지도) |
| 공지사항/정보 게시판(리스트·상세·첨부·검색) | `board.html` | 탭(공지/정보/보도) + 검색 + 리스트 + 상세 펼침 + 첨부 |

| 회원가입·로그인(이메일) / 마이페이지 | `member.html` | 로그인·회원가입·마이페이지 탭 통합(데모 클라이언트 검증) |
| 관리자 백오피스(대시보드·회원·콘텐츠·문의) | `admin.html` | 가입자/문의 현황 대시보드, 회원 상태변경, 공지 등록(웹에디터 흉내), 문의 답변 |
| 보안/인증(단방향 암호화·XSS/SQLi 방어) | 전 페이지 카피 + README | 데모는 목 데이터, 실제 구축 시 백엔드 적용 명시 |

## 4. 데모사이트 종합 방향
- **구조/내비게이션**: ksponco식 공공기관 포털 골격(롤링 배너 → 퀵메뉴 → 사업안내 → 지표 → 공지보드 → CONTACT → 멀티칼럼 푸터).
- **비주얼/인터랙션**: 화이트 클린 베이스 + 블루·그린 그라데이션 + 롤링 배너 + Lenis 부드러운 스크롤 + GSAP 핀/스케일 리빌 + reveal/카운트업.
- **도메인**: 체육 → 창업지원으로 모든 카피·미디어 재구성.
- **미디어**: Higgsfield MCP로 히어로 영상(창업 허브) + 섹션 이미지(멘토링/IR/입주공간/글로벌) 고품질 생성.
- **기능 데모**: 게시판·1:1 문의·회원·관리자까지 목(mock) 데이터 기반으로 동작 시연 → 실제 구축 시 React/Vue + Node/Spring + RDBMS로 확장.
