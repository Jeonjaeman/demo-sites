# 대한창업지원협회 (KSSA) — 창업 지원 협회 홈페이지 데모

창업기업 대상 공지·정보 전달 및 1:1 소통을 위한 **협회 공식 홈페이지 제안용 데모사이트**.

- **콘텐츠/기능 골격**: RFP(요청사항) 기반 — 메인(롤링 배너·퀵메뉴) · 협회소개 · 공지/정보 게시판 · 1:1 문의(비밀글) · 회원/마이페이지 · 관리자 백오피스
- **디자인/UX/인터랙션/스크롤**: 국민체육진흥공단(ksponco.or.kr) 식 **라이트 공공기관 포털**(솔리드 GNB · 메인비주얼 롤링 배너 · 통합검색 · 바로가기 · 공지보드 우선)을 창업지원 도메인으로 재구성. 다른 데모(운송포탈 등)와 **의도적으로 구분되는 독자 톤·구조·폰트(Archivo)·인터랙션**으로 설계
- **미디어**: Higgsfield AI 생성 (시네마틱 히어로 영상 + 섹션 이미지 — 창업 허브/멘토링/IR/입주공간/글로벌)

> 분석 근거는 [`분석/참고사이트_분석.md`](분석/참고사이트_분석.md) 참고.

## 실행 방법
정적 사이트입니다. 프로젝트 폴더에서 로컬 서버를 띄워 여세요(영상/CDN 때문에 `file://` 대신 http 권장):

```bash
python -m http.server 8830
# 브라우저에서 http://127.0.0.1:8830/index.html
```

## 페이지 구성
| 파일 | 내용 | RFP 매핑 |
|------|------|----------|
| `index.html` | 메인 랜딩 — **메인비주얼 롤링 배너(영상+이미지 자동순환)**, **통합검색 + 바로가기 퀵메뉴**, **공지/정보/보도 탭 프리뷰**, **지원사업 카드**, 통계 카운트업, 협력기관, CONTACT | 메인(롤링배너·퀵메뉴·최신공지·시각효과) |
| `about.html` | 협회 소개 — 인사말, 비전·핵심가치, **연혁 타임라인**, **오시는 길(약도)** | 협회 소개(인사말·연혁·오시는 길) |
| `board.html` | 공지·정보 게시판 — 탭(공지/정보/보도), **검색**, 리스트, **상세 펼침**, **첨부파일**, 페이지네이션, FAQ | 공지/정보 게시판(리스트·상세·첨부·검색) |
| `inquiry.html` | 1:1 문의 — 내 문의 내역(**답변완료 상태**), 작성 폼(**비밀글**·첨부·**개인정보 동의**) | 1:1 문의(비밀글·첨부·답변상태) |
| `member.html` | 회원 — **이메일 로그인/회원가입**(검증), **마이페이지**(내 정보 수정·1:1 문의 내역) | 회원가입·로그인·마이페이지 |
| `admin.html` | 관리자 백오피스 — **대시보드**(가입자/문의 현황·차트), **회원관리**(상태 변경), **콘텐츠관리**(웹 에디터), **문의관리**(답변 등록) | 관리자(대시보드·회원·콘텐츠·문의) |

## 기술 스택 (데모)
- HTML + CSS(자체 디자인 시스템) + **순수 Vanilla JS (외부 모션 라이브러리 의존성 없음)**
- 인터랙션: IntersectionObserver 리빌·카운트업, 자체 구현 메인비주얼 롤링 배너(영상+이미지)
- 폰트: Pretendard(KO) + **Archivo**(EN, 대문자 헤드라인)
- 데이터는 `assets/js/data.js` 의 목(mock) — 실제 구축 시 백엔드 API/RDBMS로 대체

## 실제 구축 시 (RFP 반영)
- 제안 스택: FE React/Vue · BE Node/Spring · DB MySQL/PostgreSQL (확장성 있는 자체 프레임워크)
- 보안: 비밀번호 단방향 암호화(bcrypt 등), SQL Injection·XSS 방어, 1:1 비밀글 접근제어, 개인정보 동의 로깅
- 반응형(PC/태블릿/모바일) — 본 데모에 구현됨
- 본 데모의 `data.js` 스키마(공지·정보·문의·회원)가 ERD 설계의 출발점

## 미디어 출처
`assets/media/` 의 영상·이미지는 Higgsfield AI로 본 프로젝트를 위해 생성한 오리지널 에셋입니다(저작권 이슈 없음).
