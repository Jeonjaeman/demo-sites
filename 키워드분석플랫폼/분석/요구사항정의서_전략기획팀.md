# 전략기획팀 — 키워드 검색 도구 개발 의뢰서 (요구사항 정의서)
> 원문: `전략기획팀_키워드_검색_도구_개발_의뢰서_260608.pdf` (2026-06-08, 11p)
> 성격: **네이버 검색 데이터 중심**의 키워드 분석 도구 (참고: 키워드사운드 · 블랙키위 · 네이버센터)
> ※ 프리랜서 공고(`요청사항.txt`)는 이 기획안을 **SNS 중심으로 확장**하라는 요청 → 데모는 네이버 코어 + 4대 SNS 탭으로 둘 다 충족.

## 화면 구성 (0~6)

### 0. 검색 페이지
- 분석할 키워드 입력 후 검색 시작 → 분석 결과 페이지로 이동
- 최대 10개 키워드 동시 입력 (10개는 희망, 조율 가능) / 최소 1개
- 10개 초과 입력 제한 / 검색 버튼 클릭 시 분석 실행
- 입력 방식·세부 UI는 개발팀 제안 가능

### 1. 키워드 최초 검색일
- 입력 키워드별 최초 검색일 표시 (키워드명 + 날짜 함께)
- 여러 키워드 비교 가능
- **성인키워드 여부는 삭제**
- 유의: 데이터 수집이 **2016-01-01부터** → 그 이전 검색 키워드도 최초검색일은 2016-01-01로 표시

### 2. 최근 한 달 검색량
- 금일 제외 최근 30일, 네이버 통합검색 검색 횟수
- **테이블 형태 우선** (여러 키워드 비교·추출 용이 — 차트/카드보다)
- PC+모바일 **합산** 검색량만 표시
- 최대 10개 키워드 비교 테이블
- 합산 검색량 기준 **정렬 기능** 검토
- **엑셀 다운로드** 제공

### 3. 검색량 그래프
- 선택 기간 동안 키워드 검색량 변화, 키워드별 구분 비교
- 선택한 기간·필터에 따라 그래프와 테이블 데이터 함께 변경
- **일별 / 월별 / 연별** 단위 선택 (버튼 형태 선호)
- **기간 선택** 기능
- **성별 / 연령별 필터** (블랙키위 스타일)
- 그래프 **마우스 오버 시 키워드별 검색량 툴팁**
- PC+모바일 합산 / 검색량 **소수점 없이**
- 원본 데이터 **엑셀 다운로드**

### 4. 월별 / 요일별 검색 비율
- 월별(시즌성) + 요일별(주중·주말 패턴) 검색 비율
- 최대 10개 키워드 비교 / PC+모바일 합산
- 엑셀: **검색 수 + 비율 함께** 표시 / 비율 **소수 1자리**

### 5. 연관 키워드
- 입력 키워드별 연관 키워드 + 설정 기간 내 **총 검색량**
- **클러스터별 구분**(핵심 단어/주제 단위 묶음, 예: 크림·선크림·추천) — 기준/구현 논의 필요
- 오름차순 / 내림차순 **정렬**
- 연관 키워드 **클릭 시 해당 키워드 분석 결과로 이동**
- PC+모바일 합산 / 검색량 소수점 없이 / **엑셀 다운로드**

### 6. 성별 / 연령별 검색 비율
- 키워드별 성별(여/남) + 연령별 **비율 및 검색량**
- 연령: **5세 단위 희망**, 어려우면 10세 단위로 통일(혼용 지양)
- 최대 10개 키워드 비교 / PC+모바일 합산(어려우면 모바일 우선)
- 엑셀 다운로드 / 비율 소수 1자리 / 검색량 소수 없음

## 공통 규칙 (전 화면)
- 검색량 수치: **소수점 없이**
- 비율 수치: **소수 1자리**
- PC/모바일 구분 없이 **합산** 기준
- 각 영역 **엑셀 다운로드** 제공
- 데이터 시작점 **2016-01-01**
