## 지원 메시지 (붙여넣기용)

안녕하세요. 커뮤니티·리뷰 플랫폼 구축 경험이 있는 개발팀입니다.

공고를 검토한 뒤, 말씀만 드리는 것보다 보여드리는 게 빠르다고 판단해 **요구사항 전체를 반영한 시연용 데모를 먼저 만들었습니다.**

- 사용자 데모: https://jeonjaeman.github.io/demo-sites/salesup/
- 관리자 백오피스 데모: https://jeonjaeman.github.io/demo-sites/salesup/admin/

데모에서 바로 확인하실 수 있는 것:

1. **AI 자동답변** — 질문답변 게시판에 글을 올리면 AI가 자동으로 답변을 등록하는 전체 흐름(로딩 → 답변 → 면책 고지 → 선배 답변 유도). 실서비스에서는 OpenAI API 서버 연동으로 동일하게 동작합니다.
2. **회사 리뷰** — 관리자가 등록한 회사 검색·선택, 영업 직군 특화 5개 항목 별점(급여·인센티브 / 워라밸 / 영업문화 / 성장·커리어 / 목표·평가 합리성), 익명 리뷰, 회사별 평균 별점 통계.
3. **회원 등급 체계** — 증빙 서류 업로드 → 관리자 승인/반려 큐 (대표인증·전문가·영업인 뱃지).
4. **관리자 백오피스** — 회원/뱃지 관리, 회사 수기 등록 + 엑셀 일괄 업로드, 게시글 블라인드, 신고 처리, 메인 화면 큐레이션 수동 배치.
5. **PC/모바일 반응형** — 모바일 하단 탭바 포함 전 화면 대응.

기획 관점에서 두 가지를 미리 짚었습니다.

**① AI 답변 비용 문제** — 질문마다 무제한 GPT를 호출하면 트래픽 증가에 따라 토큰 비용(클라이언트 부담)이 계속 늘어납니다. 저희는 5중 가드레일로 월 비용에 상한을 두는 설계를 제안합니다: 사용자당 일일 질문 한도(등급별 차등) / 유사 질문 임베딩 캐시(반복 질문은 API 호출 없이 기존 답변 재노출) / 경량 모델 + 토큰 상한으로 건당 단가 최소화 / 월 예산 상한 도달 시 자동 일시중지 + 커뮤니티 답변 유도 폴백 / 관리자 대시보드 사용량·비용 리포트. 데모 관리자 대시보드에 이 위젯이 구현되어 있습니다.

**② 리뷰 신뢰성·법적 리스크** — 잡플래닛·Glassdoor식 "리뷰 1건 작성 시 전체 열람" 모델(데모 구현됨)로 초기 리뷰를 축적하고, 명예훼손 신고에 대응하는 임시조치(정보통신망법 44조의2) 상태값을 데이터 모델에 처음부터 반영합니다.

국내(블라인드·잡플래닛·캐치·리멤버·원티드)와 해외(Glassdoor·RepVue·Fishbowl·Comparably·r/sales) 10개 서비스 벤치마킹 문서와 화면 기획서를 보유하고 있으며, 계약 시 그대로 산출물로 이어집니다.

제안 스택은 Next.js(React) + NestJS(Node.js) + PostgreSQL + OpenAI API이며, 2주 기획 확정 → 4주 디자인 → 6주 백엔드·AI 연동 → 8주 QA·오픈의 공고 일정 그대로 진행 가능합니다. 감사합니다.

## 상세 제안 (선택 첨부용)

### 1. 프로젝트 이해
- 본 프로젝트는 단순 게시판 구축이 아니라 **"영업인 버티컬 커뮤니티의 콜드스타트 해결"** 프로젝트입니다. AI 자동답변은 빈 게시판 문제의 해법이고, 회사 리뷰는 체류시간·재방문의 앵커이며, 등급 뱃지는 콘텐츠 신뢰의 장치입니다. 세 축이 맞물려야 2단계(채용 플랫폼) 확장의 데이터 자산이 쌓입니다.
- 따라서 회사(companies)를 리뷰·게시글·미래 채용공고가 모두 참조하는 마스터 엔티티로 설계하고, 회사 상세 페이지에 채용 탭 확장 자리를 미리 확보합니다.

### 2. 수행 범위별 접근

| 영역 | 접근 |
|---|---|
| 기획/디자인 | 화면 설계서(Figma) — 데모가 곧 인터랙티브 와이어프레임 역할. 캐치 등 채용 플랫폼급 정보 밀도의 UI |
| 프론트엔드 | Next.js(React) 반응형 웹 — SSR로 커뮤니티 SEO 확보(검색 유입이 커뮤니티 성장의 핵심 채널) |
| 백엔드 | NestJS + PostgreSQL + Redis(캐시/세션) — 이메일 인증(SES), 비밀번호 bcrypt, 이미지 업로드(S3 + 증빙 자동 블러) |
| AI 연동 | OpenAI API 서버사이드 큐 처리(1~2분 내 등록 요건 충족) + 아래 비용 가드레일 |
| 관리자 | 회원/뱃지 승인 · 회사 관리(수기+엑셀 파싱, 2차 크롤러 대비 스키마) · 블라인드/신고 · 메인 큐레이션 배치 |

### 3. AI 자동답변 비용 통제 설계 (핵심 차별점)

| 층 | 장치 | 기대 효과 |
|---|---|---|
| 호출 차단 | 일일 질문 한도(기본 3회, 등급별 차등) + 최소 글자수/금칙어/중복 필터 | 스팸성 호출 원천 차단 |
| 캐시 재활용 | 질문 임베딩 유사도 검색 → 기존 답변 재노출 | 반복 질문 비용 0원, 축적 콘텐츠 SEO 재활용 |
| 단가 최적화 | 경량 모델 기본 + max_tokens 상한 + 질문 길이 제한 + 배치 큐 | 건당 비용 최소화 |
| 예산 상한 | 월 상한 설정, 80% 알림 → 100% 자동 일시중지 + 폴백 UI | **청구서 상한 보장, 서비스는 계속 동작** |
| 가시성 | 대시보드 토큰/비용/캐시 적중률 리포트 | 클라이언트가 직접 모니터링·조정 |

- 운영 안정화 후 "전체 자동 답변 → AI 답변 요청 버튼(옵트인)" 전환 옵션으로 상시 비용 추가 절감 가능.

### 4. 일정 (60일, 공고 마일스톤 준수)
- W1~2 기획 확정(화면 설계서) → W3~4 디자인(Figma) → W3~6 백엔드·API·AI 연동(병행) → W5~7 프론트 퍼블리싱/연동 → W7~8 QA·시큐어 코딩 점검·오픈
- 주 1회 정기 리포트 + 스테이징 서버 상시 공개로 진행 상황을 화면으로 확인시켜 드립니다.

### 6. 데모 체험 가이드
1. 데모 로그인(원클릭) → "AI 질문답변" 탭 → "콜드콜", "이직 연봉" 등 키워드로 질문 등록 → 3초 후 AI 답변 자동 등록(실서비스 1~2분)
2. 회사리뷰 → 회사 선택 → 두 번째 리뷰부터 잠금 확인 → 리뷰 작성 시 전체 열람 해제
3. 마이페이지 → 증빙 업로드 → 뱃지 신청 → 관리자 데모의 "회원 관리"에서 승인 처리
4. 관리자 → 회사 관리 → 엑셀 일괄 업로드 시뮬레이션 / 메인 큐레이션 순서 변경
