# 세일즈업 SALESUP — 영업인 커뮤니티 & 회사 리뷰 플랫폼 (데모)

영업 직무 종사자를 타겟으로 한 특화 커뮤니티 + 회사 별점·익명 리뷰 플랫폼의 **제안용 데모사이트**입니다.
질문 게시판 AI 자동 답변, 회원 등급(뱃지) 체계, 관리자 백오피스까지 요구사항 전 범위를 화면으로 시연합니다.

> 원 요청: [커뮤니티 및 회사 리뷰 플랫폼 반응형 웹 구축.txt](./커뮤니티%20및%20회사%20리뷰%20플랫폼%20반응형%20웹%20구축.txt) (예산 1,500만 · 60일 · 향후 채용 플랫폼 확장 1단계)

로컬 실행: 정적 파일이므로 아무 웹서버로 열면 됩니다. 예) `python -m http.server` 후 `http://localhost:8000/index.html`

## 체험 시나리오 (데모 포인트)

1. **AI 자동답변** — 로그인(원클릭) → "AI 질문답변" 탭 → 글쓰기에 "콜드콜", "이직 연봉", "미팅" 등 키워드 포함 질문 등록 → 3초 후 AI 답변 자동 등록 (실서비스 기준 1~2분, OpenAI API 연동 지점).
2. **회사 리뷰** — 회사리뷰 탭 → 회사 카드 클릭 → 두 번째 리뷰부터 잠금(기브앤테이크) → "리뷰 쓰고 전체 열람" → 영업 특화 5축 별점 입력 → 잠금 해제 + 내 리뷰 게시.
3. **뱃지 신청 → 관리자 승인** — 마이페이지에서 증빙 업로드 후 신청 → [admin.html](admin.html) 회원 관리에서 승인/반려.
4. **운영 시연** — 관리자에서 엑셀 일괄 업로드(파싱 시뮬레이션), 게시글 블라인드, 신고 임시조치, 메인 인기글 순서 변경.

## 기술 구성 (데모)

- 순수 HTML/CSS/JS 정적 사이트 — 빌드 불필요, 반응형(모바일 하단 탭바)
- 백엔드 시뮬레이션: 사용자 데이터(글·댓글·리뷰·좋아요·열람권)는 **localStorage 지속**, 관리자 상태는 인메모리
- AI 답변: 키워드 매칭 사전 + 폴백 (실서비스: OpenAI API 서버사이드 호출 + 큐 처리)
- 모든 회사·회원·리뷰 데이터는 가상입니다

실서비스 제안 스택: Next.js + NestJS + PostgreSQL + OpenAI API (기획_설계.md 5장 참조)
