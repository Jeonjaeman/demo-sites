# 한울로지스 (HANUL LOGISTICS) — 운송 포탈 데모

물류/운송 기업 대외 영업 채널 및 문의 접수 홈페이지 **제안용 데모사이트**.

- **콘텐츠/기능 골격**: 다불러 · 공유화물 벤치마킹 (운임조회 / 견적접수 / 실시간 배차현황 / 다채널 상담 / 공지·FAQ)
- **디자인/UX/인터랙션/스크롤**: 우정항공(woojungair.com/intro.do) 톤앤매너를 육상 운송 도메인으로 재구성
- **미디어**: Higgsfield AI 생성 (시네마틱 히어로 영상 + 섹션 이미지, 트럭/도로/물류센터)

> 분석 근거는 [`분석/참고사이트_분석.md`](분석/참고사이트_분석.md) 참고.

## 실행 방법
정적 사이트입니다. 프로젝트 폴더에서 로컬 서버를 띄워 여세요(영상/CDN 때문에 `file://` 대신 http 권장):

```bash
python -m http.server 8810
# 브라우저에서 http://127.0.0.1:8810/index.html
```

## 페이지 구성
| 파일 | 내용 | RFP 매핑 |
|------|------|----------|
| `index.html` | 메인 랜딩 (시네마틱 스크롤) — 히어로 영상, 브랜드 리빌(pin&scale), 강점, 공감→솔루션, 서비스 타임라인, **간이 운임조회 위젯**, **실시간 배차 현황**, 통계 카운트업, 네트워크, 뉴스, CONTACT | 메인/소개, 운임조회, 다채널 상담(플로팅) |
| `rate.html` | 간이 운임조회 풀페이지 + 차량 제원표 + 요금 산정 기준 | 간이 운임조회 |
| `quote.html` | 운송 문의/견적 접수 폼 (기업·화물·상하차지·파일첨부·**개인정보 수집동의**·접수번호 발급) | 견적 접수 폼 |
| `board.html` | 공지사항(카테고리 필터) + FAQ(아코디언) | 공지/FAQ 게시판 |
| `admin.html` | 관리자 백오피스 — 대시보드, **접수·견적 관리(상태 변경)**, **기준 단가표 엑셀 업로드 + 매트릭스 미리보기**, 공지 관리 | 관리자 페이지 |

## 기술 스택 (데모)
- HTML + CSS(자체 디자인 시스템) + Vanilla JS
- GSAP + ScrollTrigger (스크롤 핀/스케일/패럴럭스), Lenis (부드러운 스크롤)
- 폰트: Pretendard(KO) + Montserrat(EN, 대문자 헤드라인)
- 데이터는 `assets/js/data.js` 의 목(mock) — 실제 구축 시 백엔드 API/RDBMS로 대체

## 실제 구축 시 (RFP 반영)
- 제안 스택: FE React/Vue · BE Node/Spring · DB MySQL/PostgreSQL
- 향후 배차/정산 확장을 고려한 RDBMS 설계, 관리자 인증/접근제어, 개인정보 동의 로깅
- 본 데모의 `data.js` 스키마(단가 매트릭스·접수·공지·FAQ)가 ERD 설계의 출발점

## 미디어 출처
`assets/media/` 의 영상·이미지는 Higgsfield AI로 본 프로젝트를 위해 생성한 오리지널 에셋입니다(저작권 이슈 없음).
