# 참고 사이트 분석

> 운송 포탈 데모사이트 구축을 위한 벤치마킹 분석
> - **다불러 / 공유화물** → 내용 구성 · 기능 (집중 분석)
> - **우정항공(우정에어)** → 디자인 · UI/UX · 인터랙션 · 스크롤 효과 (디자인 레퍼런스)

## 1. 다불러 (dabuller.com) — 내용/기능 분석

### 1-1. 포지셔닝
"IT기반 B2B 물류" — 17년 운영 노하우 + 벤처기술을 결합한 기업 화물 배차 플랫폼. 제조업 물류 담당자를 핵심 타깃으로 한 **신뢰·전문성** 소구형 랜딩.

### 1-2. 메뉴(IA) 구성
| 대분류 | 하위 |
|--------|------|
| 회사소개 | CEO인사말 · 연혁 · 서비스 제안서 |
| 전국운임 | 전국운임 조회 · 요금 적용기준 |
| 차량제원 | 차량 제원 · 상자·파렛트 계산기 |
| 소식 | 공지사항 · 언론보도 · 인증현황 |
| 인재채용 | 인재채용 |
| 상담채널(상시노출) | 카카오톡 상담 · 1800-0510 전화 · 배차프로그램 로그인 |
| 견적문의(CTA) | 일반 견적문의 · 기업 정기계약(월 정산) |

### 1-3. 핵심 기능
1. **전국 운임 조회** — 지도에서 지역 클릭 → "5초 만에" 예상 운임. 회원가입 불필요.
2. **견적 문의 2종** — ①단건 일반화물 ②기업 정기계약(월 단위 물량·전담).
3. **배차 프로그램(SaaS)** — 화주 직접 사용 가능. 실시간 추적·정산·통계 자동 집계, PC/모바일.
4. **실시간 배차 현황 테이블** — 접수시각/업체명/화물정보/수배차종/운송구간/진행상태 (LIVE, 5초 자동 갱신).
5. **상담 채널 3종** — 카카오톡(평균 1분 응답)·전화(07~22시 연중무휴)·프로그램.
6. **퀵툴 5종** — 서비스 제안서(PDF) · 운임조회 · 요금 적용기준 · 차량 제원표 · 상자/파렛트 계산기.

### 1-4. 콘텐츠 스토리텔링 흐름 (랜딩 섹션 순서)
1. 히어로(가치제안 + CTA + 신뢰배지)
2. 운임조회 인터랙션 + 핵심지표(17년 / 2,100+ 기업 / 7개 정보망 / 1만+ 카톡)
3. **공감(이런 고민 있으셨나요? 6가지)** → **솔루션(다불러는 이렇게 해결)**
4. 접수 채널 소개(카톡/전화/프로그램)
5. CORE SERVICES — 7개 정보망 통합 매칭 엔진 설명
6. MATCHING ALGORITHM — 4가지 매칭 기준(단가/거리/차량적합/검증)
7. 운영방식 2종(전국 콜배차 / 고정노선)
8. 실시간 배차 현황(LIVE)
9. 요금 체계 3종(협의형/고정형/예치형)
10. 고객사 로고 슬라이드(2,100+) + 업종현황
11. 성장 지표(연매출/배차건 그래프) + 공식 인증
12. 퀵툴 5종 → 브랜드 스토리 → 본사 → 푸터(상담 CTA 고정)

### 1-5. 데모에 차용할 포인트
- **"공감→솔루션" 카피 구조**가 매우 강력 (담당자 고민을 직접 인용)
- 운임조회를 **랜딩 상단 인터랙션**으로 끌어올림
- 숫자 지표·인증배지로 신뢰 구축
- 상담채널/견적 CTA를 **상시 고정 노출**

## 2. 공유화물 (1800-9924.com) — 내용/기능 분석

### 2-1. 포지셔닝
"화물판 카카오택시" — GPS 기반 실시간 자동배차 + 합배송으로 물류비 절감. **비회원 즉시 주문 / 10초 견적**의 간편함을 전면에 내세움. (다불러보다 대중·소량·즉시성 지향)

### 2-2. 메뉴(IA) 구성
온라인주문 · 주문내역 · 다마스용달 · 라보용달 · 1톤용달 · 퀵서비스 · 차량종류 · 기업 월거래 · 즐겨찾기
(우측 고정: 적재물보험 2억 · Google Play · App Store · 블로그 / 좌하단: 배차 프로그램)

### 2-3. 핵심 기능
1. **온라인 주문 위젯(메인 히어로 좌측)** — 지도 + 차량선택 + 출발지/도착지 주소검색 → 단계별 진행 → 예상요금(원) 실시간 표기. 22:00~06:59는 예약접수.
2. **실시간 주문 현황 테이블** — 상호(성명)/톤수/차량/출발지/도착지/상태(완료·대기·예약).
3. **AI 자동 견적·배차** — 출발/도착 입력 시 즉시 요금 산출 + 기사 배차.
4. **차량종류 안내** — 오토바이~25톤, 탑차/윙바디/장축/리프트/냉동차 등 차종별 제원·적재량 상세표.
5. **기업 월거래** — 정기/긴급/전국망, 기업 전용 대시보드(주문내역·정산·세금계산서).
6. **챗GPT Q&A** — 자주 묻는 질문(물류비 절감·운송비 변경 요인·요금 계산방식·산재보험 등)을 아코디언으로 제공.
7. **AI 추천 화물운송 리스트** — 지역·차종별 실시간 주문 노출.
8. **앱(Android/iOS) 연동 · 카카오톡 알림 · 신용카드 결제 · 적재물보험 2억**.

### 2-4. 데모에 차용할 포인트
- **온라인 주문/운임조회 위젯을 히어로 핵심 요소로 배치**(지도+차종+주소→요금)
- **실시간 주문/배차 현황 테이블**로 활성도·신뢰감 연출
- **차량 제원 상세 안내**(데모의 "간이 운임조회"·"차종 선택"과 직결)
- 비회원 즉시 접수 등 **진입장벽 최소화** 메시지

## 4. 우정항공 (woojungair.com/intro.do) — 디자인/UX/인터랙션/스크롤 레퍼런스

> 데모사이트의 **비주얼·인터랙션 톤앤매너는 이 사이트를 카피**한다. (콘텐츠/기능은 위 1·2에서, 디자인은 여기서)

### 4-1. 디자인 시스템
- **무드**: 다크 시네마틱 + 클린 화이트의 교차. 프리미엄 물류 기업 톤.
- **컬러**: 메인 블루 그라데이션(시안 `#3DBDF0` → 딥블루 `#0A2A66` 계열), 네이비 텍스트(`#1B2333`), 화이트/라이트그레이 배경, 포인트 그라데이션 원형 아이콘 배지.
- **타이포**: 영문 대문자 **초대형 볼드 헤드라인**(Hero "BEYOND EXPECTATION…"), 한글은 가는 본문. 섹션 타이틀 뒤에 옅은 **워터마크 대형 텍스트**.
- **로고 모티프**: 새의 날개(그라데이션). 인터랙션 시 날개 로고가 반복 등장.

### 4-2. 섹션 구조 (스크롤 순서)
1. **인트로 스플래시** — 좌하단 `SKIP` 버튼 → 전체화면 인트로 영상 후 진입.
2. **Hero** — 풀스크린 배경 **영상**(물류 터미널/지게차 루프) + 다크 오버레이 + 초대형 영문 헤드라인 + 한글 서브카피 + 하단 스크롤 다운 셰브론.
3. **스크롤 전환(핀+스케일)** — 스크롤 시 히어로 영상이 **둥근 모서리 프레임으로 축소**되고, 배경에 대형 워터마크 로고/사명이 드러남 (GSAP ScrollTrigger pin/scale 류).
4. **강점 카드 그리드** — "20년 전문성 / 글로벌 허브 인프라" 등 카드가 **fade-up + stagger**로 등장, 그라데이션 원형 아이콘.
5. **SERVICE INFO** — 세로 **타임라인형 지그재그** 레이아웃. 좌우 교차 이미지 카드(블루 그라데이션 오버레이 + 라운드), MORE 필 버튼, 가운데 세로선 + 날개 로고가 스크롤 따라 이동.
6. **NETWORK** — 다크 네이비 섹션, **회전하는 3D 도트 지구본**, 지역 탭(미주/유럽/아시아), 파트너 로고 그리드 reveal.
7. **NEWS** — 라이트 섹션, 뉴스 카드(카테고리/제목/요약/날짜) fade-up.
8. **CONTACT US** — 블루 + 아이소메트릭 기하 배경, 초대형 화이트 헤드라인 + CONTACT 필 버튼.
9. **Footer** — 다크 네이비, 로고 + 회사정보 + 멀티컬럼 사이트맵 + 약관/개인정보 버튼 + Related Site 드롭다운.

### 4-3. 인터랙션 / 스크롤 효과 (데모에 구현할 목록)
- **부드러운 스크롤**(Lenis 류) + 스크롤 기반 애니메이션(ScrollTrigger 류)
- 히어로 영상 **pin & scale-down → 프레임화**
- 섹션 진입 시 **fade-up / stagger reveal**
- 섹션 타이틀 뒤 **워터마크 텍스트 패럴럭스**
- **카운트업** 숫자(지표), 회전 지구본, 탭 전환
- **헤더**: 최상단 투명 → 스크롤 시 솔리드/축소
- **Back-to-top** 플로팅 버튼, **스크롤 다운 셰브론** 바운스
- 카드/버튼 hover 마이크로 인터랙션(필 버튼 화살표 이동)

### 4-4. 영상·이미지 품질 방침 (사용자 요구 "영상·사진 품질 신경")
- Hero 배경 영상: 실사급 물류 현장(트럭/터미널/도로 주행) **시네마틱 루프** — Higgsfield 생성.
- 서비스/네트워크 이미지: 고해상도, 블루 그라데이션 오버레이로 톤 통일.
- 모든 생성 미디어는 **운송(트럭/도로/물류센터) 도메인**으로 — 우정항공은 항공이지만 본 프로젝트는 **육상 운송**.

## 5. 데모사이트 종합 방향
- **콘텐츠/기능 골격**: 다불러의 "공감→솔루션→채널→서비스→지표→CTA" 스토리 + 공유화물의 "히어로 운임조회 위젯 + 실시간 현황 테이블".
- **비주얼/인터랙션**: 우정항공의 다크 시네마틱 + 블루 그라데이션 + 스크롤 핀/리빌/패럴럭스.
- **도메인 치환**: 항공 → 육상 화물 운송(트럭/도로/물류센터)으로 모든 카피·미디어 재구성.
- **미디어**: Higgsfield MCP로 히어로 영상 + 섹션 이미지 고품질 생성.
