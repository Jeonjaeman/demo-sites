# MOA — 앱 커뮤니티 + 자체 광고 플랫폼 (제안용 데모)

> 앱 내 **자체 광고 시스템**과 사용자가 직접 글을 쓰는 **커뮤니티 게시판**을 통합한 플랫폼의 제안용 데모입니다.
> 요청사항([`요청사항.txt`](요청사항.txt)) 기반으로, Supabase·Next.js·Flutter 웹뷰 구성을 **정적 HTML/CSS/JS 목업**으로 시연합니다.
> 다른 데모(운송포탈·창업지원협회)와 **독립된 폴더**이며 의존하지 않습니다.

## 화면 구성 (3 Surfaces)

| 파일 | 화면 | 핵심 |
|------|------|------|
| [`index.html`](index.html) | 플랫폼 소개(랜딩) | 3개 화면 진입 · 기능 하이라이트 · 기술 스택 |
| [`app.html`](app.html) | **사용자 커뮤니티**(Flutter 웹뷰) | 익명글·댓글/대댓글·투표·스포일러·번역·좋아요/스크랩·신고 + 5종 광고 + 실시간 트래킹 로그 |
| [`admin.html`](admin.html) | **관리자 어드민** | 4단계 권한·제재 · 게시글(HTML 에디터/예약발행) · 신고 처리 · 캠페인/5종 소재 · 트래킹 통계·CSV |
| [`advertiser.html`](advertiser.html) | **광고주 대시보드** | 본인 캠페인 현황 · 소재별 성과 · 노출/클릭/CTR 추이 · CSV |

## 요청사항 → 데모 매핑

- **게시판/회원 관리자** — 4단계 권한(관리자/광고주/유료/무료) 승인·반려, 회원 제재(차단), 게시글(공지/광고/일반) HTML 에디터·예약발행, 댓글·신고 처리 → `admin.html`
- **광고 캠페인·통계** — 캠페인 생성/편집(기간·플랜·타겟·상태·자동 만료), 5종 소재(피드/팝업/배너/바텀시트/슬라이드), 클릭 리다이렉트, 노출/클릭/CTR 집계·CSV → `admin.html`
- **광고주 대시보드** — 로그인 광고주 본인 캠페인 통계·CSV → `advertiser.html`
- **사용자 게시판** — 익명 닉네임·500자·이미지 3장, 댓글·대댓글·신고, 스포일러, 투표글, 좋아요·스크랩·내 글/스크랩, 클라이언트 AI 번역(원문 보기), SEO 메타 → `app.html`
- **비기능** — 노출/클릭 Batch 트래킹(우측 로그 시각화), AI 번역 Edge Function 프록시(코멘트로 명시)

## 기술 스택 (실제 구축 기준)

- **백엔드**: Supabase (PostgreSQL · Auth · Storage · Edge Functions)
- **프론트엔드**: Next.js / React (SEO 최적화 · OG · JSON-LD)
- **연동**: Flutter 앱 WebView 브릿지
- **광고 트래킹**: 노출/클릭 이벤트 Batch 적재로 DB 부하 최소화 + 캠페인 자동 만료
- **백오피스 UI**: 상용 UI 템플릿(MUI 등) 활용 전제 (본 데모는 자체 디자인 시스템으로 시연)

## 로컬 실행

```bash
# 레포 루트에서
python -m http.server 8810
# http://127.0.0.1:8810/커뮤니티광고플랫폼/            → 플랫폼 소개
# http://127.0.0.1:8810/커뮤니티광고플랫폼/app.html     → 사용자 커뮤니티
# http://127.0.0.1:8810/커뮤니티광고플랫폼/admin.html   → 관리자 어드민
# http://127.0.0.1:8810/커뮤니티광고플랫폼/advertiser.html → 광고주 대시보드
```
