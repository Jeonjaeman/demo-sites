[프로젝트 개요]
앱 내 정해진 위치에 노출될 자체 광고 시스템과 사용자가 직접 글을 작성하는 커뮤니티 게시판을 통합 구축하는 프로젝트입니다. 상용 UI 템플릿을 활용한 관리자용 웹 어드민 및 광고주용 통계 대시보드 구축과 Flutter 앱 내 웹뷰(WebView)로 연동될 사용자 게시판 프론트엔드 개발이 포함됩니다.

[프로젝트 배경 및 목표]
앱 내 커뮤니티 기능을 강화하여 유저 체류 시간을 늘리고, 자체적인 광고 캠페인 운영을 통해 수익화 기반을 마련하고자 합니다. 관리자와 광고주가 사용하는 백오피스 영역은 상용 UI 템플릿을 적극 활용하여 디자인/퍼블리싱 비용을 절감하고, 일반 유저가 사용하는 커뮤니티 게시판과 광고 트래킹 등 핵심 기능 구현에 예산을 집중하여 Supabase 기반으로 효율적으로 구축하는 것이 목표입니다.

[과업 범위]
1. 수행 범위
- 기획 및 디자인: 사용자 게시판(웹뷰) 화면 설계 및 디자인, 웹 어드민/광고주 대시보드는 상용 UI 템플릿(MUI 등) 활용하여 구성
- SW 개발: 프론트엔드 웹 개발, Supabase 백엔드 연동, 광고 트래킹 로직 구현

2. 상세 기능 요구 사항
2-1. 게시판 및 회원 관리자
- 4단계 사용자 권한 관리 (관리자/광고주/유료/무료) 및 권한 신청 승인/반려
- 전체 회원 목록 조회 및 불량 유저 제재(차단) 관리 기능
- 게시글(광고/공지/일반) 작성 HTML 에디터, 복수 이미지 업로드, 게시 상태(예약발행 등) 관리
- 유저가 작성한 게시글 및 댓글 목록 관리, 유저 신고 내역 접수 및 처리 기능
2-2. 광고 캠페인 및 통계 관리
- 광고 캠페인 생성/편집 (기간, 플랜, 타겟, 상태 관리, 자동 만료 처리)
- 5종 광고 소재 관리 (피드 카드형, 전면 팝업형, 하단 배너형, 바텀시트형, 하단 슬라이드형)
- 광고 클릭 시 외부 링크로 연결하기 위한 리다이렉트(Redirect) 라우팅 로직
- 광고 노출수/클릭수/CTR 집계 그래프 및 CSV 내보내기
2-3. 광고주 대시보드
- 광고주 로그인 및 본인 캠페인 집행 현황 통계 조회, CSV 내보내기
2-4. 사용자 게시판
- 관리자 작성 게시글(공지/광고) 노출 및 일반 유저 글 작성 (익명 닉네임, 500자, 이미지 3장)
- 모바일 환경을 고려한 이미지 업로드 전 클라이언트 단 리사이징/압축 처리
- 게시글 내 댓글 작성, 대댓글, 악성 유저/게시글 신고 기능
- 스포일러 기능 (비로그인 시 일부 가림 처리) 및 투표글 기능
- 좋아요, 스크랩, 내 글/스크랩 보기 기능
- 클라이언트 단 AI 기반 번역 기능 (캐시, 로딩, 재시도, 원문 보기 지원 / DB 저장 안 함)
- 게시글 SEO 메타정보 설정 (URL Slug, OG Image, JSON-LD 등)

3. 비기능적 요구사항
3-1. 성능/규격: 광고 노출/클릭 트래킹 시 Supabase DB에 과부하가 가지 않도록 최적화된 로직(Batch 등) 설계 필요
3-2. 보안/인증: AI 번역 API 호출 시 클라이언트 단에서 API Key가 노출되지 않도록 Supabase Edge Functions 등을 활용한 프록시 처리

[기술/제조 스택]
- 백엔드: Supabase (PostgreSQL, Auth, Storage, Edge Functions) 필수 사용
- 프론트엔드: Next.js, React 등 (SEO 최적화가 가능한 프레임워크 협의 가능)
- 연동 환경: Flutter 앱 내 웹뷰(WebView) 연동 고려

[클라이언트 준비 사항]
- 문서 및 자료: 요구사항 정의서 및 대략적인 화면 구성 아이디어
- 투입 인력 및 조직: 프로젝트 매니저 및 Flutter 앱 개발자 (웹뷰 브릿지 연동 협업)

[주요 일정]
- 희망 착수일: 계약 완료 후 1~2주 이내
- 주요 마일스톤: 착수 후 3주 내 어드민 템플릿 세팅 및 캠페인 기능 구현, 6주 내 사용자 게시판(댓글/신고 포함) 및 광고 소재 구현, 8주 내 트래킹/통계 및 QA 완료
- 최종 오픈(납품) 희망일: 착수일로부터 2개월 이내

[지원 자격 및 우대 사항]
- 지원 자격: Supabase 기반의 백엔드 연동 경험 및 Next.js/React 프론트엔드 개발 경험이 있는 기업 또는 팀
- 우대 사항: 자체 광고 시스템(노출/클릭 트래킹) 구축 경험자, Flutter 웹뷰(WebView)와의 브릿지 통신 연동 경험자, 관리자 UI 템플릿 커스터마이징에 능숙한 파트너 우대

[산출물]
- 기획/디자인 원본 파일 (사용자 게시판 영역)
- 프론트엔드 소스코드 원본 및 Supabase 스키마/Edge Functions 코드
- 시스템 운영 및 관리자 매뉴얼
